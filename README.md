# ConsoleLocalServer

Основаная Информация:

Язык: **C#**

Устройсва подтерживающие работу: **[Windows7]** **[Windows10]** **[Windows11]**

Версия: 0.1

Лецензия: 
Программа ConsoleLocalServer -программа для офисного, ученческого использования. 

Программа - позволяет открывать локальные сервева для контроля устройсв. 

Программа - бесплатна, и не требует ни упаминания, ни оплаты за работу. 

Программа - не Хакерская приблуда, а средство котроля устров для Офисов и Учереждений для обучения 



*Как использовать?:*
1) Устоновите программу на все нужные вам устройсва.
2) Настройте и откройте сервер.
3) Запустите программу на других устройсвах
4) Подключитесь к Серверу.
Поздровляю! Теперь вы можете следить за устройсвами.
